# 介绍

* 这是一个笔记仓库, 记录了学习数据结构的过程
* 使用JAVA语言从底层实现了主流且重要的数据结构

## 实现的数据结构如下

* 动态数组
* 链表, 双向列表
* 栈, 队列, 堆, 优先队列
* 二分搜索树
* 线段树, 字典树
* 平衡树: AVL-Tree, 红黑树(添加操作)
* 哈希表
